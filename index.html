Taxi Trip Duration Prediction

The goal is to create a model that, given information like the pick-up and drop-off locations or the time and date, can precisely forecast how long a trip would take. This challenge is to precisely forecast the length of taxi rides in New York City (NYC)."Train.csv" and "Test.csv," two datasets are provided for the challenge. The training set is "train.csv." It will be used to build and train the model and contains ground-truth labels. There are examples in "test.csv" without labels. Kaggle keeps these labels secret. The same properties as the training set are used to identify each sample, which is then identifiable by a special ID number. We achieved an accuracy of 85%, which is excellent..
